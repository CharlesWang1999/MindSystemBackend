# 视频数据存储约定

```text
Data
└─video_year_month_day_hour_minue_second
   └─round
      ├─kinect_action.avi
      └─camera_micro_expression.avi
```

For example:

```text
Data
├─video_2022_09_10_08_20_10
|  ├─01
|  |  ├─kinect_action.avi
|  |  └─camera_micro_expression.avi
|  ├─02
|  |  ├─kinect_action.avi
|  |  └─camera_micro_expression.avi
|  └─03
|     ├─kinect_action.avi
|     └─camera_micro_expression.avi
├─video_2022_09_10_08_50_18
|  ├─01
|  |  ├─kinect_action.avi
|  |  └─camera_micro_expression.avi
|  ├─02
|  |  ├─kinect_action.avi
|  |  └─camera_micro_expression.avi
|  └─03
|     ├─kinect_action.avi
|     └─camera_micro_expression.avi
...
```
